# Flow_Assignment

# 빌드 시 필요한 설정이 있습니다.
local.properties 내에 API 값을 설정해야 합니다.
```properties
~
api.id="{api id 값}"
api.pw="{api pw 값}"
```
필요한 값은 Flow 업무 댓글을 참고 부탁드립니다.
